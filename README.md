# 🌦️ Weather App

A responsive Weather App built using **HTML**, **CSS**, and **JavaScript** that provides real-time weather information through the **OpenWeatherMap API**. The application allows users to either search for weather by city name or fetch weather details for their current location using the browser's Geolocation API.

## 🚀 Features

- Search weather by city name
- Get weather using current location
- Displays temperature, humidity, wind speed, and cloudiness
- Dynamic weather icons and country flag
- Loading indicator while fetching data
- Stores user coordinates using Session Storage
- Responsive and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- Geolocation API
